# Rugby Contextual Model

This repository contains the data and R code associated with the study:  
**"The Influence of Contextual Variables on Key Performance Indicators and Player Rankings in Elite Rugby Union"**

The goal of this research is to evaluate how contextual factors (Match Location, Team Quality, and Opponent Quality) impact Key Performance Indicators (KPIs) and subsequent player rankings in professional rugby.

---

## 📂 Repository Structure

* `Rugby_KPI.R`: The primary R script used for data processing, statistical modeling, and generating player rankings.
* `Model plots/`: Contains the visual outputs (coefficient plots, ranking comparisons) generated by the analysis.
* `data/`: *(Action required: Upload your dataset here)* Raw or processed match-day data used for the model.
