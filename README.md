# Rugby Contextual Model

This repository contains the data and R code associated with the research paper:  
**"The Influence of Contextual Variables on Key Performance Indicators and Player Rankings in Elite Rugby Union"**

The study evaluates how variables impact Key Performance Indicators (KPIs) and subsequent player rankings.

---

## Repository Structure

* **`Raw_Data/`**: Contains the match-day datasets
* **`Model plots/`**: Visual outputs and coefficient plots generated from the mixed-effects models.
* **`Rugby_KPI.R`**: The primary R script for data processing, statistical modeling, and generating context-adjusted player rankings.
